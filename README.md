
# 🌐 WebChange Detector

🚀 **Live Demo:** https://webchange-detector-ten.vercel.app/

A modern web application to **monitor website changes in real-time**.
Track updates, detect differences, and stay informed automatically.

---
## 🎥 Project Demo Video
👉 [Watch the Demo](https://drive.google.com/file/d/17n0cVxAfFyDcWl7KTFirNI9SYoTwBiOc/view?usp=drive_link4)

-----

## ✨ Features

* 🔍 Monitor multiple websites
* 📊 Detect and track content changes
* 🧠 Smart diff view for comparing updates
* ⚡ Fast performance using Vite
* 📱 Fully responsive UI

---

## 🛠️ Tech Stack

* ⚛️ React
* 🟦 TypeScript
* ⚡ Vite
* 🎨 Tailwind CSS
* 🧪 Playwright (testing)

## 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/webchange-detector.git
cd webchange-detector
```

---

### 2️⃣ Install dependencies

```bash
npm install
```

---

### 3️⃣ Run the app

```bash
npm run dev
```

Open in browser:

```
http://localhost:8080/
```
